# Frontend Dashboard (Structure)

This lightweight dashboard provides a base structure for:
- admin KPI cards,
- interactive traffic map placeholders,
- real-time feed panel (WebSocket integration hook).

## Run

Open `frontend/dashboard/index.html` in a browser.
